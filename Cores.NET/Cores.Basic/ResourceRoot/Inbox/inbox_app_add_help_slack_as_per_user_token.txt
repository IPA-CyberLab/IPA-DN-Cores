1. https://slack.com/signin にアクセスし、一度ワークスペースにログインする。

2. 1 と同一のブラウザで、https://api.slack.com/custom-integrations/legacy-tokens にアクセスする。

3. 「Legacy tokens」メニューが表示される。

4. 画面をスクロールし、「Legacy token generator」を表示する。

5. 目的の Workspace に対する Token が未生成 (none) の場合は、「Create token」ボタンをクリックしてトークンを生成する。

6. 生成されたトークン ("xoxp-" で始まる文字列) をコピーする。

7. 以下のフォームの「Client Secret」欄に、6 でコピーしたトークン ("xoxp-" で始まる文字列) を貼付けする。
※ 「Client ID」は空欄のままでよい。
